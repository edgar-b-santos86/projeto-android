Conteúdos Periféricos

Conteúdos periféricos são aqueles que não estão diretamente relacionados ao conteúdo principal da página, mas que são conteúdos que podemos achar importantes ou interessantes para o usuário como, curiosidades, links que também contenham informações que podem complementar o conteúdo, redes sociais e mais.

Nosso conteúdo periférico está inserido em nosso <aside></aside> e vamos fazer algumas configurações bem interessantes para que ele fique destacado como um conteúdo extra.

A principio a primeira configuração do <aside></aside> foi a seguinte:

                
                    aside {
                        background-color: var(--cor1);
                        padding: 10px;
                        border-radius: 10px;
                        box-shaow: 3px 3px 8px rgba(0, 0, 0, 0.2);
                    }

Essa configuração foi feita a principio para mostrar quer a lista criada no aside estava com os marcadores fora do quadro/caixa do aside e isso é necessário configurar também as listas dentro dele e o código ficou assim:

                
                    aside > ul {
                        list-style-type: '\2714\00A0\00A0';
                        list-style-position: inside;
                        columns: 2;
                    }

O que fizemos no código acima foi trocar a imagem do marcados de listas com a declaração list-style-type, e nos valores colocamos o código numérico do marcardo (sem o U+)que encontramos no Emojipedia, além disso colocamos a contra barra ("\") na frente do código e também inserimos o NBSP, que é o espaço sem quebra de linha.

Colocamos também a declaração list-style-position com o valor inside isso fez com que os marcadores ficassem dentro do bloco do <aside></aside>.

Temos dentro do nosso aside um título e também queremos que ele tenha algo diferente dos demais, sendo assim foi feita a seguinte configuração para o h3 do aside:

                
                    aside > h3 {
                        background-color: var(--cor4);
                        color: white;
                        padding: 10px;
                        margin: -10px -10px 0px -10px;
                        border-radius: 10px 10px 0px 0px;
                    }

Para que a configuração ficasse devidamente alocada precisamos que a margem do h3 ficasse para fora, pois quando criamos uma margem ela é teoricamente interna, e isso faria com que a margem do h3 deixasse sobrar a bargem do aside o que fucaria muito estranho.

Para isso usamos a declaração margin: -10px -10px 0px -10px;, o que fez com que a margem superior ficasse negativa, vazando para fora, e assim o h3 ficasse colado no topo do aside. Depois apenas colocamos um border-radius: 10px 10px 0px 0px sem arrendondar as partes inferiores esquerda e direita para que assim pudessemos ter um resultado onde paraceria que tinhamos criado um único bloco.

Uma outra coisa que criamos nos site é que todos os links terão uma diferenciação para os links internos, neste caso, colocamos a figura de elos de corrente utilizando o mesmo método que usamos para os marcados da lista do nosso aside.

A unica coisa que fizemos foi criar uma classe para links externos para que pudessemos diferencia-los dos demais links e usamos um pseudo-eletemento. Nas CSS o código ficou dessa forma:

                
                    a.externo::after {
                        content: '\00A0\1F517'
                    }
