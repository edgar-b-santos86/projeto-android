Usando Variaveis em CSS (Capitulo 17 Aula 8)

Aqui vamos falar um pouco sobre variaves, sim, variaveis em CSS.

Dentro do arquivo CSS geralmente os desenvolvedores colocam as cores de sua paleta por exemplo em um comentário dentro do código e quando vão utilizá-las copiam e colam para a aplicação isso para a cor da fonte, fundo da página, enfim, fazem dessa forma, entretanto, fazendo isso com as variáveis nós podemos facilizar o uso e até mesmo a troca de cores quando quisermos mudar a cor do nosso site ou se tivermos construído ao para o nosso cliente e ele desejar fazer alterações.

Abrindo nosso arquivo CSS faremos o seguinte nós colocaremos a pseudoclasse :root, ela é a raiz do código e todas as variáreis que colocarmos nela poderemos usar em qualquer marcação do código.

Toda variavel em CSS começa com o sinal de dois traços "--" e a declaração dela fica assim:

                
                    :root {
                        --cor1: #f0f0f0;
                        --cor2: #dcdcdc;
                        --cor3: #c0c0c0;
                        --cor4: #a9a9a9;
                        --cor5: #04304d;
                    }

Acima declaramos as variaveis da nossa paleta de cores, e dessa forma poderemos usar para configurar as cores de qualquer texto ou área do nosso site na nossa CSS.

Para usar as variaveis declaradas acima, basta colocarmos o nome da variavel dentro da propriedade que queremos configurar, por exemplo:

                
                    body {
                        background-color: var(--cor0);
                        color: var(--cor5);
                    }

Acima configuramos a cor de fundo do body e a cor do texto, e para isso usamos a propriedade background-color e color, e dentro delas colocamos o nome da variavel que declaramos anteriormente.

Para usar as variaveis em qualquer lugar do código, basta usar o nome da variavel dentro da propriedade que queremos configurar, e assim podemos usar as variaveis para cores, fontes, tamanhos, etc.

Para usar uma fonte, por exemplo, podemos declarar a variavel da seguinte forma:

                
                    :root {
                        --font-padrao: Arial, Verdana, Helvetica;
                        --font-destaque: Merienda, cursive;
                    }

Depois para usar a fonte padrão do site, basta colocar dentro da propriedade font-family:

                
                    body {
                        font-family: var(--font-padrao);
                    }

Lembrando que o nome da variavel não pode ter espaços.

Com isso podemos usar as variaveis para facilitar a manutenção do código e também para facilitar a troca de cores e fontes quando necessário.

Criando Regras

Agora que já sabemos como usar as variaveis, vamos criar algumas regras para o nosso site.

Podemos criar regras para que conseguiremos incluir mais configurações dentro das propriedades do nosso site, uma das que usamos na página do desafio foi a @font-face, e uma outra que sempre utilizaremos é a @charset, e para a importação de fonts usamos a @import.
