EXPLICAÇÃO DO DESAFIO

Aqui teremos todas as Explicações de como o desafio foi proposto e resolvido passo a passo.

Melhorando o Formato do Conteúdo (Capitulo 17 Aula 10)

Configurando os Parágrafos

No desafio começamos pela configuração do parágrafo, pois quando fizemos a configuração globral de margin 0px e padding 0px nós eliminamos a margem natural que todos os parágrafos possuem e neste caso no desafio foi proposto que fizessemos o seguinte dentro da CSS:

                
                    main p {
                        margin: 15px 0;
                        text-aling: justify;
                        text-indent: 30px;
                        font-size: 1em;
                        line-height: 1.4em;
                    }

Dessa forma alteramos nos parágrafos, a margem superior e inferior (margin: 15px 0px), alinhamento do texto (text-aling: justify;), identação ou espaço do início do parágrafo (text-ident: 30px;),tamanho da fonte (font-size: 1em;), e altura da linha para o espaçamento entre uma linha e outra (line-height: 1.4em;).

Uma outra configuração que foi realizada na página do desafio foi nas palavras que foram envelopadas na tag <strong></strong> que ficaram assim:

                
                    main strong {
                        color: var(--cor4);
                        font-weight: bold;
                    }

Configurando os Links

Agora vamos configurar os links, começando com os externos, e na CSS as declarações ficaram assim:

                
                    main a {
                        text-decoration: none;
                        font-weight: bold;
                        color: var(--cor5);
                        background-color: var(--cor1);
                        padding: 2px 6px;
                    }

                    main a:hover {
                        text-decoration: underline;
                        color: var(--cor4);
                    }

Configurando os títulos

Agora vamos alplicar as alterações em nossos títulos colocamos as seguintes configurações na CSS:

                
                    main h2 {
                        font-family: var(--font-android);
                        color: var(--cor4);
                        font-size: 1.3em;
                        background-image: linear-gradient(to right, var(--cor1), transparent);
                        text-indent: 8px;
                    }
